How to use

                    System control:
                    Motors can be controlled, using 2 parameters: speed (RPM) and frequency (Hz), which are marked on sliders.
                    
                    To control the motor(s), press the following buttons:
                    1. Control with Modbus for running the motors from the website
                    2. Shutdown
                    3. M1/M2 - each motor will be run with a dedicated frequency generator
                        M1||M2 - motors will be run using one frequency generator
                    4. Clockwise/Counter-Clockwise, for setting the motor vector
                    5. Adjust either the frequency slider or speed slider
                    6. Observe!
                    
                    To stop the motor(s)
                    1. Set sliders to 0
                    2. Shutdown
                    3. Stop
                    
                    Use the overview tab to see the motors parameters without risk of changing them.
